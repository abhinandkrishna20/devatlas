### Vite

Vite is a frontend build tool.

Advantages:

- Fast startup
- Fast HMR
- Better than Create React App

### React + TypeScript

Created project using:

npm create vite@latest devatlas -- --template react-ts

### React Router

Installed:

npm install react-router-dom

Purpose:

- Client-side routing
- Multiple pages without full reload
